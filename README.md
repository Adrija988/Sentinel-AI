# Sentinel AI

Sentinel AI is an AI-powered surveillance system designed to detect weapons and violent activities in real time. The project combines a YOLOv8-based weapon detection model and a MobileNetV3-based violence detection model with a React frontend and Python backend.

## Features
- Violence activity recognition
- Live video stream analysis
- Modern React-based dashboard
- Fast and accurate deep learning models

- Python
- YOLOv8
- TensorFlow / Keras
- MobileNetV3
- React.js
- Vite

## Model Weights
Due to GitHub file size limits, `Weapon_best.pt` and `Violence_Detection_best_model.keras` are hosted on Kaggle.

Download:
https://www.kaggle.com/datasets/adrija1802/combined-model

Place the downloaded model file in the backend folder of the project directory before running the application.
